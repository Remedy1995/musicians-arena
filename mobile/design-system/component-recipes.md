# Mobile Component Recipes

## `TalentCard`

Use for discovery feeds and search results.

### Structure
- Hero image or media preview
- Name and primary role
- City and region
- Rating and completed jobs
- Skill chips
- Price range
- CTA row: `Message` and `Book`

### Visual Notes
- Use large rounded corners
- Let the name breathe
- Keep metadata compact and readable

## `GigCard`

Use in the gig board.

### Structure
- Event type badge
- Title
- Date and time
- City and venue summary
- Budget range
- Required talent chips
- Urgent badge when applicable
- CTA: `Show Interest`

### Visual Notes
- Urgent gigs should have a stronger top accent
- Budget should be easy to spot

## `BookingSummaryCard`

Use in booking detail and payment review.

### Structure
- Status badge
- Event title
- Date and venue
- Talent/client summary
- Agreed amount
- Deposit and balance

## `NotificationRow`

Use in notification inbox.

### Structure
- Event icon
- Title
- One-line body
- Relative timestamp
- Unread dot

## `MessageBubble`

### Sender Bubble
- Ember or dark filled
- White text

### Receiver Bubble
- Soft stone surface
- Dark text

### System Bubble
- Centered small capsule
- Muted background
- Low emphasis text
