# Mobile Screen Blueprints

## 1. Home

### Sections
- Greeting header
- Global search
- Featured talents
- Open gigs near you
- Recommended categories

### Primary Feeling
Curated and alive.

## 2. Explore Talents

### Sections
- Search field
- Filter chips
- Sort control
- Scrollable talent cards

### Primary Feeling
Easy comparison without clutter.

## 3. Talent Profile

### Sections
- Hero media
- Name, role, verification
- Rating and reliability metrics
- Bio
- Portfolio carousel
- Skills and event types
- Price information
- Sticky action bar

### Primary Feeling
Premium personal brand page.

## 4. Gig Board

### Sections
- Search and filters
- Open gigs
- Urgent gigs
- Matching gigs

### Primary Feeling
Opportunity feed with strong signal density.

## 5. Gig Detail

### Sections
- Title and urgency
- Organizer info
- Event details
- Requirements
- Budget
- Matching skills summary
- CTA: `Show Interest`

### Primary Feeling
Clear, decisive, opportunity-focused.

## 6. Chat

### Sections
- Participant header
- Booking or gig context strip
- Messages
- Composer

### Primary Feeling
Fast and human, not noisy.

## 7. Booking Detail

### Sections
- Status timeline
- Event summary
- Payment summary
- Notes and negotiation history
- Message shortcut

### Primary Feeling
Structured and trustworthy.

## 8. Notifications

### Sections
- Unread summary
- Today
- Earlier

### Primary Feeling
Action-oriented and lightweight.
